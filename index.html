Course Overview

This 3-week course teaches HTML and CSS from fundamentals to responsive web design with examples and exercises.

  • Week 1: Programming fundamentals & HTML basics
  • Week 2: CSS syntax, Box Model, Flexbox
  • Week 3: Advanced CSS, Grid & responsive design

Week 1 — Programming Fundamentals + HTML

Class 1: Intro to Programming Concepts

Learn logic, variables, loops, and conditionals. Set up tools: VS Code, Git, and GitHub.

  • Install VS Code & Git
  • Create a GitHub repository
  • Push your first HTML file

Class 2: HTML Structure

Understand <!DOCTYPE>, <html>, <head>, and <body>. Learn headings, paragraphs, links, lists, and images.


<h1>Hello World</h2>
<p>Welcome to HTML</p>

Class 3: Semantic HTML & Accessibility

Use <header>, <footer>, <nav>, and forms with labels and alt text for accessibility.

Week 2 — CSS Basics

Class 4: CSS Syntax, Selectors, Colors & Fonts

Learn inline, internal, and external CSS. Experiment with colours and fonts.

body { color: #333; font-family: Arial; }

Class 5: Box Model

Understand content, padding, border, and margin. Explore box-sizing: border-box.

Class 6: Flexbox Fundamentals

Use Flexbox for alignment and spacing.

.nav { display: flex; justify-content: space-between; }

Week 3 — Advanced CSS & Responsive Design

Class 7: Positioning & z-index

Control element positions using static, relative, absolute, and fixed.

Class 8: CSS Grid Basics

Build two-dimensional layouts using Grid.

Class 9: Responsive Design

Adapt designs for all devices with media queries (mobile-first approach).
